0x00 create_function()简介

适用范围：PHP 4> = 4.0.1，PHP 5，PHP 7

功能：根据传递的参数创建匿名函数，并为其返回唯一名称。

语法：

string $args 声明的函数变量部分

string $code 执行的方法代码部分
